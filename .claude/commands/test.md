# Run Tests

Execute the test suite for the project.

## Instructions

1. **Backend Tests**: Run pytest in the backend directory
   ```bash
   cd backend && python -m pytest -v
   ```

2. **Frontend Linting**: Run ESLint on the frontend
   ```bash
   cd frontend && npm run lint
   ```

3. Report any failures with file paths and line numbers
4. If tests fail, suggest fixes based on the error messages
